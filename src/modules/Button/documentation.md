#### A basic button

> A basic button that works standalone and in forms.

> To submit a form the button must have `type` "submit".

<Example>
  <Button>Button</Button>
</Example>

#### A big button

> Buttons currently has two sizes controlled by prop `big`.

<Example>
  <Button big>Button</Button>
</Example>
